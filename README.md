# 🏠 AI-Powered Real Estate Booking Agent

An intelligent Real Estate Booking Agent built using **Microsoft Copilot Studio**, **Power Automate**, and **Microsoft Dataverse** to automate property viewing requests and booking management.

## 📌 Project Overview

The Real Estate Booking Agent helps customers book property viewings through a conversational AI experience. The agent collects customer requirements, schedules appointments, triggers automated workflows, and stores booking details in Dataverse.

This project demonstrates how AI agents can streamline customer interactions and automate business processes within the Microsoft Power Platform ecosystem.

- 🤖 Conversational AI-powered property booking

## 📋 Workflow

### Step 1: Customer Interaction
The AI agent welcomes the customer and collects:

- Property Type
- Number of Bedrooms
- Customer Name
- Email Address
- Preferred Viewing Date

### Step 2: Data Validation
The agent validates all customer inputs before processing the booking request.

### Step 3: Power Automate Trigger
A cloud flow is triggered from Copilot Studio.

### Step 4: Dataverse Storage
Booking details are stored in the Dataverse table.

### Step 5: Confirmation
The customer receives a booking confirmation message.
